-----------ПРОГРАММА ДЛЯ УЧЁТА ЗАДАЧ И УПРАВЛЕНИЯ ИМИ-----------------

Запуск программы осуществляется через NiTask.exe

ФУНКЦИОНАЛ МЕНЮ
Нажатие на кнопку "МЕНЮ" - открывается основной функционал.

СОЗДАТЬ ЗАДАЧУ - кнопка для добавления задачи в список. В диалоговом окне устанавливаем сроки к задаче, категорию, приоритет, пишем подробное описание.
После добавления/завершения задачи необходимо нажать на кнопку "обновить" для того, чтобы увидеть результат

СОЗДАТЬ КАТЕГОРИЮ - кнопка вызывает диалоговое окно для создания категории задач. Название категорий - на усмотрение пользователя

ПРОСМОТРЕТЬ КАТЕГОРИИ - кнопка вызывает окно для просмотра категорий и управления ими. Неиспользуемую категорию можно удалить, однако если в ней есть задачи - сделать это не получится.

ВЫПОЛНЕННЫЕ ЗАДАЧИ - кнопка инициализирует окно со списком выполненных задач. В этом списке их можно просмотреть и при необходимости удалить ненужные.

ОБНОВИТЬ - кнопка обновляет данные в таблице с задачами после добавления/завершения задачи.

ПОДРОБНЕЕ О ЗАДАЧЕ - Для открытия окна с информацией о задачи, необходимо дважды нажать на ячейку с названием. Также можно выбрать одним нажатием и в меню кликнуть "подробнее".

УДАЛЕНИЕ ЗАДАЧ/КАТЕГОРИЙ - Для удаления категории или выполненной задачи, необходимо выбрать ячейку с названием и нажать удалить, затем подтвердить.


В левом нижнем углу находятся уведомления о сроках, обращайте на них внимание!
